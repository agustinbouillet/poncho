Plantillas

Login y Registro

Login

Página inicial que muestra primero la forma de acceder a la plataforma a usuarios que ya se registraron, recuperar su contraseña y crear una nueva cuenta.

Registro

Página donde el usuario crea una nueva cuenta con la menor cantidad de datos posibles.

Correo de bienvenida

Correo electrónico que le confirma al usuario que se creó una nueva cuenta.

Recuperar contraseña (envío)

Página que pide el correo electrónico del usuario para iniciar el proceso de recuperar contraseña.

Recuperar contraseña (envío exitoso)

Página de respuesta de la página anterior (recuperar contraseña).

Correo de recuperar contraseña

Correo electrónico con el enlace para acceder a la página para finalizar el proceso de recuperar contraseña.

Recuperar contraseña (generar nueva)

Página para finalizar el proceso de recuperar contraseña.
